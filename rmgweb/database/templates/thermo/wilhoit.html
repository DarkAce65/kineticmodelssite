{% comment %} This template expects a variable thermoData as a list in the following form: thermoData[0] = The value 'Wilhoit' thermoData[1] = Heat capacity at zero temperature in J/mol*K thermoData[2] = Heat capacity at infinite temperature in J/mol*K thermoData[3] = Wilhoit a0 parameter thermoData[4] = Wilhoit a1 parameter thermoData[5] = Wilhoit a2 parameter thermoData[6] = Wilhoit a3 parameter thermoData[7] = Enthalpy at zero temperature in kJ/mol thermoData[8] = Entropy at zero temperature in J/mol*K thermoData[9] = Intermediate temperature B in K thermoData[10] = Minimum valid temperature in K thermoData[11] = Maximum valid temperature in K All of these values should be strings in the desired format. {% endcomment %}
